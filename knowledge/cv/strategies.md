### Priority Repositories

**Tier 1 - High Impact**
1. **uniswap/uniswap-v3-core** - Core DeFi protocol
2. **pancakeswap/pancake-frontend** - Already have contributions
3. **curvefi/curve-contract** - Major DeFi

**Tier 2 - Good for Visibility**

6. **transmissions11/solmate** (Popular Solidity lib)

**Tier 3 - Beginner Friendly**
7. **primitivefinance/interface**
8. **rxr-labs/one-inch-liquidity-protocol**

### How to Contribute

1. **Find Issues**
   - Filter by "good first issue" label
   - Look for "help wanted" tags
   - Check bounty boards (Gitcoin, Layer3)

2. **Before Starting**
   - Read contribution guidelines
   - Check existing PRs/issues
   - Comment "I'd like to work on this"

3. **Make the PR**
   - Fork the repo
   - Create feature branch
   - Write tests
   - Document changes
   - Submit with clear description

4. **After PR**
   - Respond to feedback
   - Make requested changes
   - Follow up if stale

### Tracking Contributions

Maintain a log of:
- Repositories contributed to
- Issues/PRs opened
- PRs merged
- Bounties claimed
